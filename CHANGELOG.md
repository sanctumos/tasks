# Changelog

## [Unreleased]

### Added

- **Project CRUD, membership, pins, to-do lists, task linkage** ‚Äî `GET/POST` directory project update, project members, `todo_lists` + `tasks.list_id`, `user_project_pins`, client-only directory filter (`client_visible` + `person_kind=client`), `backfillTaskProjectIdsFromLegacyNames()` in PHP. Tasks: `project_id` / `list_id` on create, update, and list filters; `directory_project` on task JSON when joined. New admin page `workspace-project.php` (from `workspace-projects.php`). Idempotent DDL in `config.php` / `api_python/db.py` and migration notes updated.
- **Organizations & project directory** ‚Äî SQLite `organizations`, `projects`, `project_members`; `users.org_id` and `users.person_kind` (`team_member` \| `client`); `tasks.project_id` (nullable FK) for later backfill from legacy `tasks.project` string. Default org **Default** + user org linking on bootstrap. API: `GET /api/list-organizations.php`, `GET /api/list-directory-projects.php`, `POST /api/create-directory-project.php`. PHP + FastAPI mirror; SDK `list_organizations`, `list_directory_projects`, `create_directory_project`; `create_user` accepts optional `org_id` / `person_kind`. Admin: **Users** shows org + person kind; **Workspace projects** (`/admin/workspace-projects.php`) lists and creates directory projects. `createUser` self-heals default org before insert. Idempotent migration notes in `tools/migrations/sanctum_001_organizations_and_projects.sql`. **Deploy:** server must `git pull` (or your usual sync) to this `main` ‚Äî the browser UI will not change until the new PHP is on the host.
- **Heartbeat setup wizard** ‚Äî `scripts/setup_heartbeat.sh`: interactive bash wizard to configure an open-claw heartbeat worker. Asks for agent name, sanctum folder, Tasks creds (type now or path to existing .env), heartbeat project, optional worker user id, interval, and whether to add a crontab entry. Writes `run_heartbeat.sh` (bash + curl) and `.env.heartbeat` under `sanctum/agents/<name>/`. See [docs/HEARTBEAT.md](docs/HEARTBEAT.md) and [docs/HEARTBEAT_WIZARD_CONTEXT.md](docs/HEARTBEAT_WIZARD_CONTEXT.md).
